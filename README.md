Currency_Exchange_Bot.
Телеграм-бот для конвертации валют. Имя в телеграме t.me/AnatoleCurrencyBot.

Этот телеграм-бот знает несколько команд:

/start или /help: выдает стартовое сообщение и пишет формат запроса;
/values: выписывает все валюты, которые знает бот;
<валюта 1> <валюта 2> <количество> выводит значение количества валюты 2 от валюты 1.
Данный бот не запущен, необходимо запустить бот со своего устройства. Для работы с данным телеграм-ботом необходимо установить пакеты requests и PyTelegramBotAPI.
